# ROOM_404

## Overview
大学生の男の子がクラスメイトに怖い話を披露する話。

## Plot
大学一年生の**ぼく**はひょんなことから参加した**みんなで怖い話をする会**で、クラスメイトの**長谷川美樹**の気を引くために、自分が住むマンションを舞台とした「４０４号室」という怪談を創作する。目論見どおり長谷川と距離を縮めることに成功したぼくだったが、自分が創作したとおりの出来事が現実に起きてしまい、追いつめられていく。
